= Nettoyer les configurations VASA dans ONTAP tools
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Pour mener à bien le processus de nettoyage VASA, suivez ces étapes.


NOTE: Il est recommandé de supprimer tous les datastores vVols avant de commencer le nettoyage VASA.

.Étapes
. Annulez l'enregistrement du plug-in en accédant à \https://OTV_IP:8143/Register.html
. Vérifiez que le plug-in n'est plus disponible sur vCenter Server.
. Fermez les outils ONTAP pour VMware vSphere VM.
. Supprimez les outils ONTAP pour VMware vSphere VM.
