= Réinitialiser le mot de passe utilisateur de la console de maintenance
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Lors du redémarrage du système d'exploitation invité, le menu GRUB affiche une option permettant de réinitialiser le mot de passe de l'utilisateur de la console de maintenance.  Utilisez cette option pour mettre à jour le mot de passe utilisateur de la console de maintenance sur la machine virtuelle.  Après avoir réinitialisé le mot de passe, la machine virtuelle redémarre pour définir le nouveau mot de passe. Dans un scénario de déploiement haute disponibilité, après le redémarrage de la machine virtuelle, le mot de passe est automatiquement mis à jour sur les deux autres machines virtuelles.


NOTE: Pour les ONTAP tools for VMware vSphere HA, vous devez modifier le mot de passe utilisateur de la console de maintenance sur le nœud de gestion des outils ONTAP , qui est node1.

.Étapes
. Connectez-vous à votre serveur vCenter
. Cliquez avec le bouton droit de la souris sur la machine virtuelle et sélectionnez *Power* > *Restart Guest OS*
Pendant le redémarrage du système, l'écran suivant s'affiche :
image:../media/maint-console-password.png["Réinitialiser le mot de passe de l'utilisateur de maintenance"]
+
Vous avez 5 secondes pour choisir votre option. Appuyez sur n'importe quelle touche pour arrêter la progression et geler le menu GRUB.

. Sélectionnez l'option *Réinitialiser le mot de passe de l'utilisateur de maintenance*. La console de maintenance s'ouvre.
. Dans la console, saisissez et confirmez le nouveau mot de passe.  Vous avez trois tentatives.  Le système redémarre après la saisie correcte du nouveau mot de passe.
. Appuyez sur *Entrée* pour continuer.  Le système met à jour le mot de passe sur la machine virtuelle.



NOTE: Le même menu GRUB apparaît également lors de la mise sous tension de la VM. Toutefois, vous ne devez utiliser l'option de réinitialisation du mot de passe qu'avec l'option *Redémarrer le système d'exploitation invité*.
