= Réinitialiser le mot de passe utilisateur de l'application
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Suivez ces étapes pour réinitialiser le mot de passe utilisateur de l'application nécessaire à l'enregistrement du fournisseur SRA et VASA auprès de vCenter Server à l'aide des ONTAP tools for VMware vSphere.

.Étapes
. Ouvrez un navigateur web et accédez à : `\https://<ONTAPtoolsIP>:8443/virtualization/ui/`
. Connectez-vous en utilisant les identifiants d'administrateur configurés lors du déploiement des outils ONTAP .
. Dans la barre latérale, sélectionnez *Paramètres*.
. Sur la page *identifiants VASA/SRA*, sélectionnez *Réinitialiser le mot de passe*.
. Saisissez et confirmez le nouveau mot de passe.
. Sélectionnez *Réinitialiser* pour appliquer le nouveau mot de passe.
